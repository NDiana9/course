1) Посмотреть где я - pwd
2) Создать папку - mkdir folder_name
3) Зайти в папку - cd folder_name
4) Создать 3 папки - mkdir folder1 folder2 folder3 или mkdir folder{1,2,3}
5) Зайти в любую папку - cd folder2
6) Создать 5 файлов (3 txt, 2 json) - touch file{1,2,3}.txt file{1,2}.json
7) Создать 3 папки - mkdir child_folder{1,2,3}
8) Вывести список содержимого папки - ls -lha
9) + Открыть любой txt файл - cat file_name
10) + написать туда что-нибудь, любой текст. - vim file_name --> i --> [write a text]
11) + сохранить и выйти. - esc --> :wq
12) Выйти из папки на уровень выше - cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. - mv file{1,2}.json ./child_folder3
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp child_folder3/file{1,2}.json ./child_folder1
15) Найти файл по имени -  find ../ -name find_me.raw  => ../folder1/find_me.raw
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. - tail -f file_name.txt | grep [word_to_find]
17) вывести несколько первых строк из текстового файла - head -3 long_text.txt
18) вывести несколько последних строк из текстового файла - tail -2 long_text.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. - less -s long_text.txt
20) вывести дату и время  stat -c %y long_text.txt => 2021-07-28 13:36:20.570220600 +0300
